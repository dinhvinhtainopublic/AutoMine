# 🚀 XMRig No Docker (Devbox / Codespace)

## ⚡ Quick Start

```bash
git clone https://github.com/yourname/xmrig-nodocker
cd xmrig-nodocker

chmod +x start.sh
./start.sh
```

---

## 🔧 Config

Edit file `config.sh`:

* WALLET = ví XMR
* WORKER = tên máy

## 📊 Log

Miner sẽ in trực tiếp ra terminal
